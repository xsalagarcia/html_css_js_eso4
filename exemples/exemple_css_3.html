Un altre exemple CSS

En aquest exemple es mostren novament columnes, però també altres elements, i fins i tot un efecte d'ombra i variació de mida quan s'enfoca una de les columnes.

Totes les columnes estan dins un <div>, el qual té l'atribut CSS display amb el valor flex. Això fa els elements immediatament continguts dins el div es mostrin seguin el model flexbox.

Les ombres

Les ombres poden ser interiors o exteriors. També podem modificar-ne el color.

L'atribut box-shadow serveix per donar ombra a elements que actuen de contenidors, com ara <p> o <div>, però també tenim text-shadow per donar ombra al text..

Les animacions simples

Un element pot "transicionar" de forma suau els seus atributs CSS. Per això s'utilitza l'atribut transition.. És el cas d'aquesta columna, que apareix l'ombra només quan es situa el punter del ratolí a sobre.

Els selectors CSS

Els selectors poden fer referència:

  • Un element, a través del seu nom.
  • Una classe, a través del nom de la classe precedida per un punt. Les classes s'estableixen amb l'atribut class.
  • Un identificador, que s'estableix amb l'atribut id al element, i s'hi fa referència amb el caràcter # seguit de l'identificador.
  • En aquest cas, a aquesta pàgina hi ha un sufix en un dels selectors utilitzats: :hover. Això vol dir que s'aplica el CSS allà contingut sempre que el punter del ratolí estigui soobre l'element.
